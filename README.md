How to run the website:

1. Open terminal
2. Go to project folder
3. Type "python manage.py runserver"
4. You can see a link " http://127.0.0.1:8000/ ". If you just click on it, it will show ERROR 404, be careful!
5. Add "go" to the link to open the game page (Full link --> http://127.0.0.1:8000/go)
6. Add or replace "go" with "todo" to see the update log (Full link --> http://127.0.0.1:8000/todo)
7. Enjoy!
